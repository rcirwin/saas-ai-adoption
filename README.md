# SaaS AI Adoption & Integration Planning

A marketing and landing page website for a service that helps SaaS companies adopt and integrate AI into their products.

## Project Structure

- `src/` — Website source code
- `public/` — Static public assets
- `docs/` — Service offering docs, pricing, deliverables
- `content/` — Marketing copy, messaging, value props
- `research/` — Market research, competitor analysis, AI frameworks
- `proposals/` — Client proposals and templates
